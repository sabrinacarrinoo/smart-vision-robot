# Smart Vision Robot

## Overview
Smart Vision Robot is a simulation project that combines **Computer Vision** and **basic decision-making algorithms** to control an autonomous robot in a virtual environment.

The robot analyzes its surroundings using image processing techniques and decides how to move in order to avoid obstacles.


## Features
- Real-time obstacle detection using OpenCV
- Simple decision-making logic (rule-based AI)
- 2D robot simulation environment
- Modular architecture (vision → decision → action)



## Tech Stack
- Python
- OpenCV
- Pygame
- NumPy




## How to Run

1. Clone the repository:

git clone https://github.com/sabrinacarrinoo/smart-vision-robot.git
cd smart-vision-robot


2. Install dependencies:

pip install -r requirements.txt


3. Run the project:

python main.py



## How It Works
1. The simulation generates a 2D environment with obstacles  
2. The robot captures the current frame  
3. Computer vision detects edges and obstacles  
4. A decision module determines the next move  
5. The robot moves accordingly
